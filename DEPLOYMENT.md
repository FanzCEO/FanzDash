### Kubernetes Auto-scaling
```yaml
# Already configured in k8s manifests
apiVersion: autoscaling/v2
kind: HorizontalPodAutoscaler
metadata:
  name: fanzdash-hpa
spec:
  scaleTargetRef:
    apiVersion: apps/v1
    kind: Deployment
    name: fanzdash-app
  minReplicas: 3
  maxReplicas: 50
  metrics:
  - type: Resource
    resource:
      name: cpu
      target:
        type: Utilization
        averageUtilization: 70
```
